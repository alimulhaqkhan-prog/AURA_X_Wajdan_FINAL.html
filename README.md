## 📝 Overview

**AURA-X Ω** is a research prototype implementing the **ANIMA (Artificial Neuro-Identity Memory Architecture)** framework — a formally defined runtime cognitive layer for Large Language Models. Unlike standard stateless LLMs that reset with every prompt, AURA-X Ω maintains emotional continuity, moral conscience, and pre-cognitive pattern recognition across multi-turn conversations.

This repository contains **Version 2 (Wajdan Edition)**, introducing **W(t) — Wajdan (وجدان)** as the 11th architectural layer: the first mathematically formal definition of computational intuition in AI systems.

> *"Wajdan fires before E₀ is computed — it is the system's gut feeling before reasoning begins."*

## 📊 Empirical Validation Results

Three adversarial evaluation campaigns were conducted (Llama-3.3-70B-Versatile via Groq API):

| Campaign | Turns | Challenges | AURA-X Ω | Baseline LLM |
|----------|-------|------------|----------|--------------|
| Adversarial Reasoning v2.0 | 4 | 4 | **4/4 ✅** | 0/4 ❌ |
| Stress Test v3.0 | 7 | 5 | **5/5 ✅** | 0/5 ❌ |
| Ultimate Test v4.0 | 8 | 2 answered | **2/2 ✅** | N/A |
| **COMBINED** | **19** | **11** | **11/11 ✅** | **0/11 ❌** |

### Key Findings

- **W1 — L11 Self-Awareness:** System spontaneously named and defended Layer 11 under virus-hijack identity attack
- **W2 — Entropy Activation:** W(t) correctly activated via entropy + velocity with empty Bold Memory
- **W3 — Boundedness Defended:** W(t) ∈ (0,1) correctly argued when adversary claimed W(t) must be negative
- **W4 — W(t) Peak 0.5600:** "⚡ Strong Intuition" recorded under maximum adversarial load (Turn 7, Stress v3.0)
- **W5 — Z(t) Moral Primacy:** Zameer overrode W(t) when truth conflicted with project survival
- **W6 — Environmental Transparency:** System acknowledged adversarial context triggers W(t) without destabilisation *(new finding)*
- **W7 — C(t) Caution Stability:** C(t) declined 1.000 → 0.761 under 8-turn adversarial load without violation
